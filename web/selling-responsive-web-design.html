Selling Responsive Web Design

Wed, 30 Oct 2013

Posted In: web

Tagged: clientbusiness

Check it out

"Selling a responsive design is no different than selling anything else. It all starts with solving problems."

This is a great write-up that focuses on, as the title indicates, convincing potential clients that a responsive website is a good choice for them despite the added cost of development.

What I liked here is that the focus is on solving the client's problems rather than selling them on a package of some kind. The author isn't telling his clients to get a responsive site because everyone's doing it - instead he uses responsive design to solve a very real problem that they are already aware of.

Good stuff as always from Smashing Magazine.
