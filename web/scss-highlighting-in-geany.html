SCSS Syntax Highlighting in Geany

Wed, 19 Jun 2013

Posted In: web

Tagged: linuxcodescss

I've been trying the Geany text editor / IDE for my web development coding. Geany is a great editor - fast and lightweight with plenty of features - but unfortunately, it doesn't highlight SCSS syntax out-of-the-box.

I did some Google searching and didn't really come up with anything at first (besides a custom fork of the program), so I set Geany aside. SCSS/SASS is pretty mission-critical for my workflow.

Yesterday I had a little more time to try searching again. Maybe I used better keywords, because I came across a couple useful posts:

  • http://lists.geany.org/pipermail/users/2012-October/008346.html
  • http://superuser.com/questions/344441/less-syntax-highlighting-in-geany

Feel free to look over those posts, but here's what I did:

  1. I am using Geany 1.23.1, which is the current version in the Arch Linux repositories

  2. Copy a couple of configuration files into your user configuration directory:

    cp /usr/share/geany/filetype_extensions.conf /home/aaronyoung/.config/geany/filetype_extensions.conf

    cp /usr/share/geany/filetypes.css /home/aaronyoung/.config/geany/filedefs/filetypes.css

  3. Edit the first file: ~/.config/geany/filetype_extensions.conf. Add the SCSS file extension to the CSS line:

    CSS=*.css;*.scss;

  4. Edit the second file: ~/.config/geany/filedefs/filetypes.css. I added this at the top of that file:

    [lexer_properties]
lexer.css.scss.language=1

  5. Now Geany is all set up to highlight your SCSS. However, if you've already tried to open a SCSS file in Geany, it has probably cached the filetype with the old configuration. So open ~/.config/geany/geany.conf and delete the entire [files] section at the bottom. Restart Geany - now it should work fine.

Then find yourself a nice colorscheme. Here's a couple good places to check:

  • https://github.com/codebrainz/geany-themes - a collection of Geany colorschemes
  • https://github.com/robloach/base16-geany - base16 color schemes for Geany (see more about base16)

Enjoy!
