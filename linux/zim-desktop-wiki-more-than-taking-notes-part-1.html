Zim Desktop Wiki: More than taking notes, Part 1

Wed, 09 Jan 2013

Posted In: linux

I have mentioned Zim from time to time before. This little program is probably my favorite app on my computer, and I'd like to provide a quick review / intro to Zim in this post. I use Zim for writing web design project notes, blog posts such as this one, Todo lists, documents, and for collecting information from all over. Zim is an excellent cross between Evernote and simpler programs like Notational Velocity.

In this two-part series, I'll discuss usage and functionality of Zim. This first post will deal with the "nuts and bolts" while the second part will get into some more advanced "fringe" uses of Zim that are possible because of the way Zim is built.

Update: View Part 2 of this Zim series

Zim on the Surface: Advanced Notetaking

At its core, Zim is basically a notetaking program with some advanced capabilities. Notes are stored on your computer in plain text format and are saved as you type. You can insert images and links to other files on your computer, and you can directly edit those plain text files if you wish. The text files contain wiki-formatted text that is easy to read while retaining some formatting cues (similar to Markdown text).

Zim allows for some basic formatting options such as bulleted and numbered lists; bold, italic, and underlined text, headings and paragraphs, etc.

As a Wiki, Zim allows you to easily link between pages of your notebook and can even auto-link text as you type it (for example, type :Projects:Project1 - it will automatically create the page "Projects" in your notebook, with a subpage "Project1", and the text will automatically link to that page).

At this level, you could think of Zim as a more featured version of Notational Velocity or a similar app. Of course, NV features syncing - I'll get to that later.

I'll be referring to the Zim website and documentation throughout this post, so this is a good time to start. See the Zim home page, which describes the basic idea of Zim.

How to start using Zim

  • Linux: Install zim from your package manager
  • Windows: Get the latest Zim installer from here: http://www.glump.net/software/zim (note that at this time, even though Zim 0.58 is out, the latest Windows installer is 0.57)
  • Mac OSX: See this page: http://zim-wiki.org/install.html - it appears that installation on OSX is not very straightforward. This is a shame as I feel this app would fit in really well for a lot of Mac users.

After installing, open Zim. It will ask you to specify a folder to hold your notes (Pro tip: Put them in a Dropbox subfolder! More on this later). Once you have created your notebook, Zim will open up with a fairly sparse window. Now you can start to write notes.

Write your notes. As you type, you can type the name of a new page (starting with a :colon as above) and Zim will automatically create that page. Or, right-click in the side pane and make a new page or subpage.

Start a line with "* " (asterisk - space) and the asterisk will get converted to a bullet point. You're now working in a bulleted list! Hit enter twice to get out.

Start a line with "[] " (two square brackets, then a space) and the brackets will be converted to an empty checkbox. Now you're writing a todo-list item. Click the checkbox to mark it done.

Zim has a few little tricks like this. Start working with it and you'll eventually find more.

Just below the surface: Plugins and advanced features

So, you can write notes all the time with Zim, but you might like to check the Edit->Preferences menu item and see what is available to you. Look through the options (there aren't too many), but in particular, check the "Plugins" tab. I'll highlight a few of these:

  • Distraction Free Editing - enable this and hit F11 for a fullscreen distraction-free document editor
  • Insert Screenshot - choose a menu item to take a screenshot and automatically insert that into your note
  • Journal - this used to be named "Calendar". Essentially it adds a Calendar to your notebook and allows you to create a page per day simply by clicking on a date of the calendar. Great for, well, journaling.
  • Line sorter - making a list (like this list)? Select a bunch of lines in your note and choose "Sort Lines" - they will be sorted alphabetically (good for shopping lists?)
  • Quick Note - Lets you pop up a "Quick Note" window that will automatically insert text into one of your notes. Here's how I use it: I am reading a blog post on the web, and I see some code or information I want to remember. I select it with my mouse to highlight it, then I hit a keyboard shortcut on my computer to open the Quick Note window. The Quick Note opens with my selected text already entered in - I hit "OK" and that text is added to the end of an "Inbox" note in my notebook. From there, I file it as appropriate later on. I use this a lot.
  • Tags - allows you to use @tags in your notes and provides a tag filtering function
  • Task List - manage your todos, dates, and priorities in Zim
  • Version Control - integrates with git or other version control systems. You can roll back your Zim notebook to an earlier version if you need.

Zim has a good search functionality built in as well. There is a Search menu, or you can just hit Control + Shift + F to search your entire notebook for some word. This, along with tags and the hierarchical note structure, makes it pretty easy to find what you're looking for. Zim also has keyboard shortcuts built in for many different functions to help speed up your writing.

I hope this piques your interest in this little program. At this level, I use Zim to take meeting notes, web design project notes, outline and write blog posts and documents, keep code snippets that I can copy/paste, keep a list of web bookmarks, and track other reminders and notes for myself.

Zim is a powerful and flexible program, and this post just scratches the surface of its capabilities. Keep an eye out for Part 2 of this series coming soon, where I'll dig deeper and discuss more advanced usage of Zim.
