Zim Desktop Wiki: More than taking notes, Part 2

Mon, 14 Jan 2013

Posted In: linux

Welcome back! This post will wrap up this two-part series on the Zim Desktop Wiki personal notetaking program.

In part 1, I discussed the basics of using Zim and highlighted some useful plugins that ship with Zim by default. In part 2, I will write about some useful, but not-so-obvious, features of the program.

Note: View Part 1 of this Zim series

Digging deeper: Zim behind the scenes

I've mentioned Zim being a great tool for writing. It's simple formatting options and it's overall speediness and responsiveness make it a great place to write all your notes, documents, outlines, and other material.

However, Zim can also be a great tool for collecting data, whether you're at your computer or not.

As I mentioned in Part 1, Zim stores its data in plain text files on your computer. These files contain wiki-formatted text that is easy to read and work with. This is an advantage for a few reasons:

  • You can open your notes in another program (a text editor), make a few changes, and save the file. You'll see those changes next time you open Zim.
  • You can put your Zim notebook in your Dropbox folder. This allows you to sync your notes seamlessly between various devices. I dual-boot Linux and Windows, and I have Zim installed in both operating systems and using the same notebook that is in Dropbox.
  • You can open your Zim notes in a text editor from the Dropbox app on your phone or tablet for quick reference/editing.

Automatically insert data into your Zim notes

With my Zim notes in Dropbox, I can use IFTTT (If This Then That) to insert data into my Zim notebook. IFTTT acts as "glue" between several different web services and has the ability to take data from one service, and write some text in a file in your Dropbox folder. Here are some ways I leverage that:

  • I can call a IFTTT-provided phone number with my cellphone, leave a voicemail, and it will be transcribed and added in to my Zim notes with a link to the MP3
  • I back up all my Delicious links, Tweets and Facebook statuses to Zim automatically, also with IFTTT
  • When I make online purchases and get an email reciept, I label it "Reciepts" in Gmail and it is automatically backed up in my Zim notebook

There are many other potential uses of IFTTT and Zim. The only limit is yourself.

A few more tricks

  • I save all my screenshots and scans into my Zim notebook (somewhat automatically; they are saved in the "attachment folder" for the notes).
  • I use the Quick Note plugin (pictured above) to save articles to read for later, code snippets, and other information.
  • Zim allows exporting notes to many different formats, including HTML and Markdown. This means I can save my Zim Links page to HTML and store it in Dropbox. I use it as my web browser "home page" to have quick access to all my bookmarks.
  • Because of Markdown export, I can write this blog post in Zim, export to Markdown, and use it as a new blog post in my new Pelican-based blog (which converts Markdown files to HTML).
  • Notebooks can be shared and synced with others over Dropbox (in practice, there are a couple limitations to this, notably that you can't work on the same notes at the same time).

My intent is not really to turn this post into a list of ways that I use Zim - but instead to highlight the flexibility you get when your data is stored in plain text and accessible to other programs.

Digging - yes - even deeper: Development of Zim

Zim is an actively developed program that sees minor releases every few months. These releases generally fix a few bugs and add a few new features. For example, the 0.58 release added the "Distraction Free" fullscreen writing mode I mentioned in the last post.

The project leader is responsive and open to suggestions. If you suggest a new feature for Zim (and can prove its usefulness) there's a good chance you'll get it.

To wrap up, I hope that this little series helps you get started with Zim, and I hope that Zim helps to improve your workflow. Here are a couple more reference links for you:

  • http://www.zim-wiki.org - The Zim Home Page
  • http://www.zim-wiki.org/wiki/ - The Zim Wiki (a wiki about Zim, not a Zim wiki itself). Contains tips & tricks and other information.
