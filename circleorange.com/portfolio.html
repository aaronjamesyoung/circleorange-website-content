  • Opt-e-Web

    In addition to my work at Circle Orange, I am also Website Design Director for Opt-e-Web, a small (but growing!) agency located in Salem, Oregon.

    Our current website is built on my own WordPress theme framework and uses SCSS, Zurb's Foundation framework, and some elbow grease for making the site load better on mobile devices.

  • Oregon Travel Experience

    OTE is a semi-independent state agency that manages Oregon's blue highway signs, several rest areas in the state, and the Heritage Trees and Historical Markers scattered throughout Oregon. This project was part of a large rebranding effort by OTE.

    In this project, I developed a design built for OTE by another web agency. During the course of the project, we had to make some minor changes to the design due to a shifting sense of how this website would serve OTE's mission. Other than these changes, the live website is faithful to the original design created for OTE.

    I created a custom WordPress theme, building in functionality to help OTE administer the website and increase user engagement. The theme is responsive and search-friendly.

    Code sample on Github ortravelexperience.com

  • Skyline Baptist Church

    Skyline is a medium-size Baptist church located in North Bend, Oregon. While I did not create the church logo and graphics used in the carousel area, I did design and develop a custom WordPress theme for Skyline. The theme includes an option for Skyline staff to easily switch out the "bridge" header photo without affecting the green pattern overlay.

    The site also uses WordPress for a sermon podcast and event calendar, two pieces of functionality that are essential for an active church.

    Code sample on Github sbcnb.org
